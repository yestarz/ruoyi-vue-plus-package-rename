## RuoYi-Vue-Plus包名修改器

> 支持一键修改RuoYi-Vue-Plus包名

## 使用方法：

1. 下载`RuoYi-Vue-Plus`仓库代码

打开这个页面：https://gitee.com/dromara/RuoYi-Vue-Plus
然后点击`克隆/下载`按钮，下载zip包

2. 下载本仓库代码

3. 修改本项目中的setting.properties

```properties

# 你要修改的包名
package.name=cn.xxx

# 修改后的文件路径
target.path=D:\\test

# `RuoYi-Vue-Plus`的zip包路径
zip.name=D:\\code\\RuoYi-Vue-Plus-5.X.zip

```

4. 运行`Main.java`文件
